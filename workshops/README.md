# Java Workshops

This repository contains hands-on Java workshops designed to help developers learn MongoDB through practical, real-world examples.

Each workshop includes guided exercises and focuses on a specific technology or MongoDB feature.

| Workshop | Hands-on Lab | Description | Technologies |
|----------|--------------|-------------|--------------|
| CRUD Operations with the MongoDB Java Driver | [crud-operations-workshop](https://play.instruqt.com/mongo-devrel/invite/asmaxisdfimf) | Learn the fundamentals of CRUD operations using the MongoDB Java Driver, including filtering, projections, sorting, updates, and deletes. | Java Driver |
| Aggregation Framework with Java | [aggregation-framework-workshop](https://play.instruqt.com/mongo-devrel/invite/cscphzbiq29p) | Learn how to build aggregation pipelines using the MongoDB Java Driver. Covers stages such as `$match`, `$project`, `$group`, `$lookup`, `$unwind`, `$set`, and performance best practices. | Java Driver|
| Building AI Applications with Spring AI and MongoDB | [spring-ai-workshop](https://play.instruqt.com/mongo-devrel-dev/invite/1yqfj3z8mplv) | Build AI-powered Java applications using Spring AI and MongoDB, including embeddings, vector search, RAG, and chat applications. | Spring AI |
